# Assignment 04 - React Router + Forms (JavaScript)

This archive contains a plain JavaScript (JSX) React implementation matching the assignment requirements,
plus Jest + React Testing Library unit tests for the Contact and Feedback forms.

Drop the `src/` files into a Create React App project (JavaScript) or use the structure below.

## Quick setup (recommended - Create React App)
1. Create a new CRA app (JavaScript):
   npx create-react-app my-app
2. Replace the `src/` folder in `my-app/` with the `src/` folder from this archive.
3. Install React Router and testing libs:
   npm install react-router-dom
   npm install --save-dev @testing-library/react @testing-library/jest-dom
4. Start the app:
   npm start
5. Run tests:
   npm test

Files included:
- src/*.jsx components for the app (Home, About, Contact, Feedback, AccountLayout, Profile, Settings, ThankYou, NavBar, App, index)
- Tests: src/__tests__/Contact.test.jsx, src/__tests__/Feedback.test.jsx
- index.css for basic styling
